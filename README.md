# K3D Jupyter

Jupyter notebook extension for K3D visualization library.

## Requirements

* [bower](http://bower.io/#install-bower) (to fetch K3D dependency)

## Installation

### Locally (as user)

```console
make install-user
```

### Locally (global)

```console
make install-global
```

### Using vagrant

```console
vagrant up
make install-vagrant
```
